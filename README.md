# 微博超话自动签到（浏览器扩展）

一个 Chrome / Edge 浏览器扩展：每天自动签到你关注的所有微博超话。

**零后端、零费用、零配置**——不需要服务器、不需要抓 Cookie、不需要写配置文件。安装后登录微博即可使用，所有数据只保存在你自己的浏览器里。

[English](README.en.md)

## 功能

- **自动获取超话列表**：自动翻页拉取当前账号关注的全部超话，不需要手动维护列表
- **一键签到**：遍历全部超话逐个签到，每次之间随机延迟 3–8 秒（模拟真人操作）
- **每日定时**：设定时间后每天到点自动执行，无需手动操作
- **错过补签**：设定时间时浏览器没开？下次启动浏览器时自动补签（已签过的会自动跳过，重复执行无害）
- **结果一目了然**：扩展图标显示徽章（绿色数字 = 完成数，红色 ! = 有失败），点开可看每个超话的详细结果
- **简繁双语界面**：自动跟随浏览器语言，也可手动切换

## 安装

### 方式一：应用商店（推荐）

> 上架 Edge 扩展商店审核中，链接稍后补充

### 方式二：手动加载

1. 下载本项目代码（[GitHub 下载](https://github.com/200-design/Weibo-Super-Topic-Auto-Check-in-Browser-Extension-/archive/refs/heads/main.zip)，Gitee 镜像稍后提供），解压到任意文件夹
2. 打开浏览器，地址栏输入 `chrome://extensions`（Edge 输入 `edge://extensions`）
3. 打开右上角「开发者模式」开关
4. 点击「加载已解压的扩展程序」，选择解压出来的文件夹
5. 完成！浏览器右上角会出现扩展图标

## 使用

1. **登录微博**：在浏览器中正常登录微博（扩展会自动检测登录状态，未登录时提供登录入口）
2. **手动签到**：点扩展图标 → 点「立即签到」→ 等待完成（几十个超话约需 2–3 分钟）
3. **自动签到**：勾选「每天自动签到」并设定时间，之后每天到点自动执行
4. **查看结果**：图标上的徽章提示本次结果，点开扩展可看每个超话的成功 / 已签 / 失败明细

## 权限说明（逐条解释，最小化申请）

| 权限 | 用途 |
| --- | --- |
| `storage` | 在你的浏览器本地保存设置（如自动签到时间）与最近一次执行结果 |
| `alarms` | 实现每日定时触发 |
| `scripting` + `tabs` | 签到请求必须从微博页面上下文发出（微博会校验请求来源），扩展需要在 m.weibo.cn 标签页中执行签到代码；若当时没有微博标签页，会自动开一个后台标签页并在跑完后关闭 |
| `m.weibo.cn` / `passport.weibo.com` 主机权限 | 仅访问微博相关域名，获取超话列表与发送签到请求 |

## 隐私承诺

- 不收集、不上传任何数据，没有任何外部服务器通信
- 不读取、不存储、不传输你的 Cookie——登录状态完全由浏览器自身管理
- 所有数据（设置、签到记录）只存在你本地浏览器的扩展存储中，卸载扩展即全部清除
- 代码完全开源，欢迎审查

## 限制与风险（请阅读）

- **浏览器没开就不会执行**：本扩展依赖浏览器运行。当天错过会在下次启动时补签，但如果连续多日不开电脑，中间的天数无法补救。需要「完全无人值守」请考虑云端方案
- **签到接口可能变动**：微博随时可能调整接口，若失效请提 Issue
- **使用风险自负**：本扩展模拟真人操作（随机延迟、页面上下文请求）以降低风控风险，但不能保证账号绝对安全，请自行评估

## 常见问题

**Q: 提示「未登录」怎么办？**
A: 点「打开微博登录页」登录 m.weibo.cn 后，重新打开扩展即可。

**Q: 为什么签到时会闪过一个微博标签页？**
A: 微博的签到接口要求请求从自己的页面发出，扩展需要借一个微博标签页来发请求。它是后台标签页（不会打断你），跑完自动关闭。

**Q: 会不会重复签到？**
A: 不会。已签过的超话会显示「已签」并直接跳过,重复执行完全无害。
